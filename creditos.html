Sobre mí

Hola, soy Marcelo, informático, con interés en el desarrollo web y la automatización de tareas con RPA.

Este proyecto lo realicé como parte de mi aprendizaje en HTML, CSS y JavaScript. Quise crear un juego simple, donde los usuarios puedan jugar Piedra, Papel o Tijera contra la máquina.

Fue una buena oportunidad para practicar programación, aplicar lógica de condiciones, manipular el DOM y practicar diseño web.

Herramientas utilizadas:

  • HTML5
  • CSS3
  • JavaScript Vanilla

Agradecimiento especial a los recursos y comunidades que me ayudaron en este proceso: Alura Latam, w3schools, Stark Overflow y GeeksforGeeks. También gracias a quienes prueban el juego y dejen sus comentarios.
